What's To Come:

We have lots planned for SocialIO so here is a brief outline of what we expect to add in the coming months.
We hope you're as excited as we are about the future of SocialIO!

  • Large Scale Database Integration - February 2021

    We will be integrating our front end system with the backend database on a large scale to allow for users to access the full intended functionality of the application including commenting, upvoting and downvoting.

  • Ranking - February/March 2021

    Once the database has been integrated we can begin to implement the ranking system which will give user's posts a score the user a rank within each picture category based on that score.

  • Image Recognition & Safety Infrastructure
    - March 2021

    One of the key aspects of SocialIO is it's safety infrastructure. We will be implementing image recognition software which will detect photos containing people and other innapropriate content. There will also be systems to monitor comments on posts and remove malicious comments. A user reporting system will backup this infrastructure and allow users to report any content that is missed by the aforementioned systems.

  • Rewards System - March 2021

    Another unique aspect of SocialIO is the rank and reward system, we will be designing rewards which will be recieved by the highest ranking users. We aim to continuously add new reqards and incentives to continue posting awesome photos!
